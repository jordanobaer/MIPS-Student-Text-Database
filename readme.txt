1.Download the main.asm file.
2.Add the main.asm file in the same folder as MARS, this is necessary for the file I/O.
3.Run the program in MARS.
4.The first time you run the program, a text file with the students will be created in the same folder where the main file is located.
5.Users have different commands, such as ADD, SHOW, SEARCH, REMOVE AND UPDATE.
6.First, enter a Command.
7.Then enter the information requested by the program.
8.After performing all the desired commands, user can enter QUIT.
